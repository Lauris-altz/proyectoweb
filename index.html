CSS Frameworks

TailwindCSS

Requires Node.js

Type: Utility-first CSS framework

Use: Build custom designs quickly using pre-defined utility classes.

Learning curve: Intermediate

Personal Notes

Me gusta su flexibilidad y cómo permite mantener un diseño coherente. Aunque requiere Node.js, su rapidez de prototipado compensa el esfuerzo inicial.

Bootstrap

No Node.js

Type: Responsive CSS framework

Use: Quick layout and components for web interfaces.

Learning curve: Beginner → Intermediate

Personal Notes

Ideal para proyectos rápidos. Me gusta que no necesita Node.js y funciona solo con un enlace CDN.

Static Site Generators

Jekyll

Requires Ruby

Type: Static site generator (Ruby)

Use: Build blogs and portfolios from Markdown and templates.

Learning curve: Beginner → Intermediate

Personal Notes

Me gusta su simplicidad y que se integra directamente con GitHub Pages. Ideal para proyectos pequeños.

Hugo

Standalone Binary

Type: Static site generator

Use: Fast site building with Markdown and templates.

Learning curve: Intermediate

Personal Notes

Perfecto para proyectos grandes por su rapidez. Me gusta que no requiere Node.js.

Animation Libraries

GSAP

No Node.js

Type: Animation library

Use: Smooth, performant animations for websites.

Learning curve: Intermediate → Advanced

Personal Notes

Me encanta para crear efectos visuales llamativos. No requiere instalación compleja.

Framer Motion

Requires Node.js

Type: React animation library

Use: Create dynamic, interactive UI animations.

Learning curve: Intermediate

Personal Notes

Ideal para portafolios modernos en React. Las transiciones son suaves y profesionales.

Templates & Kits

HTML5 UP

No Node.js

Type: Free HTML/CSS templates

Use: Ready-made templates for quick deployment.

Learning curve: Beginner

Personal Notes

Muy útil para inspirarse o empezar un proyecto desde una base limpia.

SaaS Tools

Hygraph

SaaS (Cloud)

Type: Headless CMS

Use: Manage and deliver structured content via GraphQL API.

Learning curve: Intermediate

Personal Notes

Ideal para proyectos donde el contenido cambia frecuentemente. Me gusta su integración con frameworks modernos.

Critical Reflection (ATELIER)

Al comparar estas herramientas, entiendo mejor las diferencias entre velocidad, flexibilidad y dependencia tecnológica. Por ejemplo, Bootstrap y GSAP funcionan sin Node.js, lo que facilita la rapidez en proyectos pequeños. En cambio, TailwindCSS o Framer Motion requieren Node.js pero permiten una personalización mucho más profunda.

Prefiero combinar herramientas simples con otras más potentes según el contexto. En mi portafolio, usaría Hugo como generador principal por su velocidad, GSAP para animaciones suaves y TailwindCSS para crear una identidad visual coherente.

En conclusión, el equilibrio entre creatividad y convención es clave: no siempre lo más complejo es lo más útil, pero conocerlo te da poder para elegir bien.
